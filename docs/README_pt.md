## 🎨 Bem-vindo à Galeria de Dashboards!

Este repositório é dedicado a compartilhar dashboards do Obsidian bonitos e funcionais que você pode facilmente usar no seu próprio cofre.

**Quão simples é?** Basta clonar este repositório, escolher o dashboard que você gosta, e copiar tanto o arquivo `.md` quanto o arquivo `.css` correspondente para o seu cofre do Obsidian. É isso! Não precisa de configuração complexa, nem conhecimento avançado.

## 1. 💻 Dashboard Início Brutalista

<div align="center">

![Dashboard-Brutalist](../images/Brutalist/Dashboard-Brutalist.png)

<p><strong>Autor:</strong> <a href="https://github.com/Inlitx">Inlitx</a></p>

</div>

Um dashboard de estilo brutalista com cores vibrantes, widget de clima, relógio e cartões interativos. Perfeito para usuários que querem uma página inicial moderna e com personalidade.

**Plugins necessários:**
· Dataview (com JavaScript habilitado)

**API necessária:**
· [OpenWeatherMap](https://openweathermap.org/api) (chave de API gratuita necessária para o widget de clima)
