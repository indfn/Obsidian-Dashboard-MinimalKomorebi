## 🎨 欢迎来到仪表板画廊！



本仓库专注于分享美观且实用的 Obsidian 仪表板，让您能够轻松将其应用到自己的知识库中。

**使用方法简单吗？** 只需克隆本仓库，选择您喜欢的仪表板，然后将对应的 `.md` 文件和 `.css` 文件一同复制到您的 Obsidian 知识库即可。就这么简单！无需复杂配置，无需高级技术知识。

## 1. 💻 Brutalist 主页仪表板

<div align="center">

![Dashboard-Brutalist](../images/Brutalist/Dashboard-Brutalist.png)

<p><strong>作者：</strong> <a href="https://github.com/Inlitx">Inlitx</a></p>

</div>

一款大胆的野兽派风格仪表板，拥有鲜艳的色彩、天气小组件、时钟和交互式卡片。非常适合希望拥有个性鲜明、现代感十足主页的用户。

**所需插件：**
· Dataview（需启用 JavaScript 支持）

**所需 API：**
· [OpenWeatherMap](https://openweathermap.org/api)（天气小组件需要免费 API 密钥）

## 3. 🌿 Komorebi 主页仪表板

<div align="center">

![Dashboard-Komorebi](../images/Komorobi/komoribe.png)

<p><strong>作者：</strong> <a href="https://github.com/Inlitx">Inlitx</a></p>

</div>

一款以自然为灵感的仪表板，拥有柔和的有机美感。布局简洁清晰，色调沉静，细节精致，专为追求宁静、无干扰工作空间的用户而设计。

**所需插件：**
· Dataview（需启用 JavaScript 支持）
· QuickAdd *(可选)*

**所需 API：**
· [OpenWeatherMap](https://openweathermap.org/api)（天气小组件需要免费 API 密钥）
